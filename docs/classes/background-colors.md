# Classes de Cor de Fundo

As classes de cor de fundo são geradas a partir do mapa semântico `$semantic-colors`. O formato é:

```
.color-background-{grupo}-{tipo}-{variante}
```

Cada cor exibida abaixo é um swatch real com o valor hexadecimal correspondente.

---

## Paleta Base — Variáveis SCSS

Estas são as variáveis brutas disponíveis via `$tbdc-{cor}-{escala}`. Use-as para criar tokens personalizados ou referenciar diretamente em SCSS.
