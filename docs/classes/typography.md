# Classes de Tipografia

Todas as classes tipográficas são geradas a partir do mapa `$typographies` no SCSS. O formato é sempre:

```
.typography-{escala}-{variante}
```

Os exemplos abaixo renderizam o texto **no tamanho real** da classe.
